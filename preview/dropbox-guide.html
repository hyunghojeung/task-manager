업체 설정으로 돌아가기

Dropbox API 키 발급 안내

Blackcopy.kr에서 첨부파일을 업로드하려면 Dropbox API 키가 필요합니다.
아래 가이드를 따라 Dropbox 계정을 만들고 API 키를 발급받으세요.

필요한 정보 3가지: App Key, App Secret, Access Token

소요 시간: 약 10분

1단계: Dropbox 회원가입

1

Dropbox 홈페이지 접속

아래 주소로 접속하세요.

https://www.dropbox.com

2

회원가입

우측 상단 "가입" 또는 "Sign up" 버튼을 클릭합니다.

이메일, 비밀번호를 입력하고 계정을 생성합니다.

Google 계정으로 간편 가입도 가능합니다.

3

요금제 선택

무료 플랜 (Basic): 2GB 제공 — 소규모 업체에 적합

Plus 플랜: 2TB, 월 $11.99 — 파일이 많은 업체에 추천

이미 Dropbox 계정이 있다면 이 단계는 건너뛰세요.

2단계: Dropbox 앱 생성

4

Dropbox App Console 접속

아래 주소로 접속하세요. (Dropbox 로그인 필요)

https://www.dropbox.com/developers/apps

5

"Create app" 버튼 클릭

파란색 "Create app" 버튼을 클릭합니다.

[ Create app 버튼 위치 - 화면 우측 상단 ]
6

앱 설정 입력

아래와 같이 선택합니다:

  • Choose an API: Scoped access 선택
  • Choose the type of access: Full Dropbox 선택
  • Name your app: 업체명 입력 (예: inche-task-manager)

"Create app" 버튼을 클릭합니다.

[ 앱 생성 화면 - API 선택, 접근 유형 선택, 앱 이름 입력 ]

3단계: App Key / App Secret 확인

7

Settings 탭에서 확인

앱 생성이 완료되면 자동으로 앱 설정 페이지로 이동합니다.

Settings 탭에서 아래 두 값을 확인하세요:

  • App key: a1b2c3d4e5f6g7h8 (예시)
  • App secret: "Show" 버튼을 클릭하면 표시됩니다
[ Settings 탭 - App key, App secret 위치 ]

App secret은 절대 외부에 공유하지 마세요. 업체 설정 페이지에만 입력하세요.

4단계: 권한 (Permissions) 설정

8

Permissions 탭 클릭

앱 설정 페이지에서 "Permissions" 탭을 클릭합니다.

아래 권한을 체크하세요:

  • files.metadata.read — 파일 정보 읽기
  • files.metadata.write — 파일 정보 쓰기
  • files.content.read — 파일 다운로드
  • files.content.write — 파일 업로드

하단의 "Submit" 버튼을 클릭하여 저장합니다.

[ Permissions 탭 - 체크할 항목 4개 ]

5단계: Access Token 발급

9

Settings 탭으로 돌아가기

"Settings" 탭을 다시 클릭합니다.

10

Access Token 생성

페이지를 아래로 스크롤하면 "OAuth 2" 섹션이 있습니다.

"Generated access token" 항목에서 "Generate" 버튼을 클릭합니다.

생성된 토큰을 복사하세요. (긴 문자열)

[ OAuth 2 섹션 - Generate 버튼과 생성된 토큰 ]

Access Token은 한 번만 표시됩니다. 반드시 복사하여 안전한 곳에 저장하세요.
잃어버리면 "Generate" 버튼을 다시 눌러 새로 발급받을 수 있습니다.

6단계: Blackcopy.kr에 입력

11

업체 정보 설정 페이지에서 입력

Blackcopy.kr에 관리자로 로그인 → 관리자 페이지 → 업체정보설정으로 이동합니다.

"Dropbox API 설정" 영역에 아래 값을 입력하세요:

  • App Key: 7단계에서 확인한 App key
  • App Secret: 7단계에서 확인한 App secret
  • Access Token: 10단계에서 생성한 토큰
  • 저장 경로: 원하는 폴더명 (예: /우리회사/첨부파일)
12

연결 테스트

"연결 테스트" 버튼을 클릭하여 정상 연결되는지 확인합니다.

"연결 성공"이 표시되면 완료입니다.

설정 완료 후 작업등록 화면에서 첨부파일을 업로드하면 자동으로 Dropbox에 저장됩니다.

요약: 필요한 정보 확인

항목 확인 위치 예시
App Key App Console > Settings a1b2c3d4e5f6g7h8
App Secret App Console > Settings > Show z9y8x7w6v5u4t3s2
Access Token App Console > Settings > OAuth 2 > Generate sl.B0abc123def456...
업체 설정으로 돌아가기

설정에 어려움이 있으시면 관리자에게 문의하세요.
